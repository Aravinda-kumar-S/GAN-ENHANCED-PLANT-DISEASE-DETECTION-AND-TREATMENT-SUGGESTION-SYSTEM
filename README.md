# 🌱 PlantVision AI
### Advanced Neural Diagnosis & Advisory System

**PlantVision AI** is a production-grade, AI-powered smart agriculture platform designed to provide real-time plant disease detection, explainable visual insights, and weather-aware decision support for farmers.

---

## 🚀 Project Identity
- **Design Language:** White, premium AI dashboard, Agriculture-themed palette.
- **Goal:** Real-time disease detection, Explainable AI (Grad-CAM), Weather intelligence, End-to-end support.

## 🛠 Tech Stack
- **Frontend:** Streamlit
- **AI/ML:** TensorFlow, Keras (EfficientNet), Google Gemini API
- **Data:** GAN-Enhanced Plant Disease Dataset

## 📂 Structure
- `app.py`: Main application entry point.
- `.streamlit/`: Configuration and Secrets.
- `requirements.txt`: Python dependencies.

## ⚡ Quick Start
1. **Install Dependencies:**
   ```bash
   pip install -r requirements.txt
   ```
2. **Run Application:**
   ```bash
   streamlit run app.py
   ```
3. **Configure Secrets:**
   Update `.streamlit/secrets.toml` with your API keys.
